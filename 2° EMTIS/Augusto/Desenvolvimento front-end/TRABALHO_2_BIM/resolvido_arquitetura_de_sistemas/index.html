  • Dentro da Arquitetura de chamada e retorno de sistemas, o que significa "request"?
    •  A pedida de dados. O sistema pede dados para o usuário para que ele dê o "response".
  • Dentro da Arquitetura de Sistemas web há um método muito utilizado ultimamente, mundialmente, para se trabalhar com sistemas. Cite qual é esta metodologia.
    •  MVC. Onde mostra como o site ficará como um plano de mesa. Dentro da sigla temos M como Model, V como Viem e C como Controller. Model representará o negócio, o acesso, aplicação e manipulação dos dados no projeto. View mostrará o resultado do model para o usuário. Controller é a camada de controle, ela é a ponte entre o Model e o View.
