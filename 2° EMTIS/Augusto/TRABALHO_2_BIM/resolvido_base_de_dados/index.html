Exercícios

  • Dados são fatos que podem ser armazenados? Sim ou Não?
    •  Sim. Dados são fatos que são armazenados no sistema, no momento em que é compartilhado se torna uma informação.
  • A base de dados é utilizada para simplificar o desenvolvimento de aplicações caracterizadas pelo uso intenso de dados? Sim ou Não?
    •  Sim. A base de dados utilizará de uma maneira mais eficiente o manipulamento de dados para lhe dar a melhor resposta do problema em que desejas solucionar.
  • A base de dados é uma coleção interrelacionada logicamente? Sim ou Não?
    •  Sim. A base de dados está sempre interligada para passar informações ou dados, onde uma informação antiga ou não pesquisada está sendo armazenada nela.
  • O que é um SGBD?
    •  SGBD ou Sistema Gerenciador da Base de Dados. É onde você utiliza programas para manipular os seus dados, desde mover ou criar novos dados.
  • O que são "meta dados"?
    •  "Meta dados" são utilizados para caracterizar um dado, exemplo temos para mim o famoso "CPF", você não irá colocar um dado em "CPF" que seja "NOME", pois não faria sentido algum.
